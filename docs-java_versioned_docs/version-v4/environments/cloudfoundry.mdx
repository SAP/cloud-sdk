[The Cloud Foundry environment](https://help.sap.com/viewer/65de2977205c403bbc107264b8eccf4b/Cloud/en-US/9c7092c7b7ae4d49bc8ae35fdd0e0b18.html) allows you to create polyglot cloud applications in Cloud Foundry.
It contains the SAP BTP, Cloud Foundry runtime service, which is based on the open-source application platform managed by the Cloud Foundry Foundation.

The SAP Cloud SDK for Java supports SAP BTP Cloud Foundry Environment out of the box.

## SAP Cloud SDK Features Supported on SAP BTP Cloud Foundry

Find below the list of features we currently support:

- ✅ Consume SAP BTP services like Destination, Connectivity, IAS, XSUAA, and others
- ✅ Multitenancy
- ✅ Resilience & Caching
- ✅ Connect to and consume services from SAP S/4HANA Cloud
- ✅ Connect to and consume services from SAP S/4HANA On-Premise
- ✅ Seamless use of typed clients provided by the SAP Cloud SDK

In most situations the SAP Cloud SDK for Java will implicitly manage complexities of this runtime for you.
