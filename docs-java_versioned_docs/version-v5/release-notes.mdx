## Should I update?

:::tip SAP Cloud SDK Version 5.0.0 is out ðŸ¥³

We are happy to announce the latest release of the SAP Cloud SDK.
SAP Cloud SDK Version 5.0.0 is the first release of the SAP Cloud SDK as an **open source** project under the Apache 2.0 license.
You can find the project sources on [GitHub](https://github.com/SAP/cloud-sdk-java).

Version 5 also updates the SAP Cloud SDK to the latest Java technology stack, inlcuding the JakartaEE APIs, Spring 6 and Java 17.
Make sure to check out the [**updated requirements**](#important-compatibility-changes-with-version-5) and the dedicated [**upgrade guide**](/docs/java/v5/guides/5.0-upgrade-steps).
:::

We highly recommend regularly updating to the latest SAP Cloud SDK version.
It will help you:

- Ensure access to the latest SAP Cloud SDK features
- Keep up with the latest changes in SAP Cloud Platform
- Update client libraries giving access to latest SAP services on SAP Cloud Platform and SAP S/4HANA
- Protect yourself from bugs and breaking changes in the future
