Interface DataAdapter

An interface with the method signatures to get exchange rates, exchange rate type details and default tenant settings. You must provide an implementation of this interface when calling the non-fixed rate Conversion APIs. This enables the library to fetch the relevant ExchangeRate, ExchangeRateTypeDetail and TenantSettings to perform conversions.

    • Defined in packages/models/src/data-adapter.ts:36
