## Public Support Channels

If your question or support incident does not contain internal or sensitive information, use the public support channels like GitHub and Stack Overflow.

### GitHub

Please create an issue in one of the public repositories:

- **Java**: ask a question, give feedback or create an issue [here](https://github.com/SAP/cloud-sdk-java/issues/new/choose).
- **JavaScript/TypeScript**: ask a question, give feedback or create an issue [here](https://github.com/SAP/cloud-sdk-js/issues/new/choose).
- **Documentation**: request documentation or suggest fixes [here](https://github.com/SAP/cloud-sdk/issues/new/choose).

### Stack Overflow

Use the `sap-cloud-sdk` tag on [Stack Overflow](https://stackoverflow.com/questions/tagged/sap-cloud-sdk) to search for solved issues or create yours.
