## Public Support Channels

### GitHub

Please create an issue in one of the public repositories:

- **Java**: ask a question, give feedback or create an issue [here](https://github.com/SAP/ai-sdk-java/issues/new/choose).
- **JavaScript/TypeScript**: ask a question, give feedback or create an issue [here](https://github.com/SAP/ai-sdk-js/issues/new/choose).
- **Documentation**: request documentation or suggest fixes [here](https://github.com/SAP/ai-sdk/issues/new/choose).
