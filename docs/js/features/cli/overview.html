What is the SAP Cloud SDK CLI?

CLI is deprecated

The SAP Cloud SDK CLI is deprecated. We'll provide replacement for key value adds of the CLI like project scaffolding, etc.

The SAP Cloud SDK CLI is a command-line interface (CLI) to simplify common tasks you may encounter when working with the SAP Cloud SDK:

  • Adding the SAP Cloud SDK to an existing project
  • Creating a new project using the SAP Cloud SDK and other recommended dependencies
  • Generate an OData client
  • Prepare your project for deployment to SAP Business Technology Platform

Installing the SAP Cloud SDK CLIâ€‹

The SAP Cloud SDK CLI can be installed with npm. We recommend installing the SAP Cloud SDK CLI globally.

$ npm install -g @sap-cloud-sdk/cli

Once the installation is complete, you can test if you have the newest version.

$ sap-cloud-sdk -v

@sap-cloud-sdk/cli/0.1.12

If you encounter any issue or want to look at the source code, look at our GitHub page.

Caret Dependencies below 1.0.0

The SAP Cloud SDK CLI is currently below version 1. Due to the behavior of global dependencies in npm, you need to run npm install -g @sap-cloud-sdk/cli@latest to update to the latest version.

Getting Helpâ€‹

You can display the list of all commands or help for a specific command directly with the SAP Cloud SDK CLI.

# Show the list of all commands
$ sap-cloud-sdk help
# Show help for the init command
$ sap-cloud-sdk help init

Autocomplete for Your Shellâ€‹

The SAP Cloud SDK CLI can provide autocomplete information to your shell.

# Print how to add autocomplete for your shell
$ sap-cloud-sdk autocomplete
