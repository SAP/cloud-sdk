`top` limits the number of returned results.
This can also be useful for paging:

<Tabs groupId="version" defaultValue="major" values={[ { label: 'SDK 2.x', value: 'major' }, { label: 'SDK 1.x', value: 'deprecated' } ]}>
<TabItem value="major">

```ts
const { businessPartnerApi } = businessPartnerService();
businessPartnerApi.requestBuilder().getAll().top(10);
```

</TabItem>
<TabItem value="deprecated">

```ts
BusinessPartner.requestBuilder().getAll().top(10);
```

</TabItem>
</Tabs>

The example above retrieves the first ten `BusinessPartner` entities.
