If you're stuck and can't find a solution to you problem, we're here to help you!

## Public support channels

### GitHub

- **Java**: Ask a question, give feedback or create an issue on [GitHub](https://github.com/SAP/cloud-sdk/issues/new/choose)
- **JavaScript/Typescript**: Ask a question, give feedback or create an issue on [GitHub](https://github.com/SAP/cloud-sdk-js/issues/new/choose)

### Stack Overflow

- Use the `sap-cloud-sdk` tag on [Stack Overflow](https://stackoverflow.com/questions/tagged/sap-cloud-sdk)

## Reporting an BCP incident for registered SAP customers

- As a registered SAP customers, report issues by creating incidents in [SAP Support Portal](http://help.sap.com/disclaimer?site=https://support.sap.com/home.html).
- Use **XX-S4C-SDK** component to reach out to us.

### How responsive are you?

- In most cases you will hear from us within 1-2 day.
- The resolution time heavily depends on the issue and can't be easily predicted.

### Who replies in support?

- Support is done by SDK developers and we take it seriously.
