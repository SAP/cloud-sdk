To replace an entity instead of just updating specific fields, use `replaceWholeEntityWithPut`:

<Tabs groupId="version" defaultValue="major" values={[ { label: 'SDK 2.x', value: 'major' }, { label: 'SDK 1.x', value: 'deprecated' } ]}>
<TabItem value="major">

```ts
const { businessPartnerApi } = businessPartnerService();

businessPartnerApi
  .requestBuilder()
  .update(businessPartner)
  .replaceWholeEntityWithPut();
```

</TabItem>
<TabItem value="deprecated">

```ts
BusinessPartner.requestBuilder()
  .update(businessPartner)
  .replaceWholeEntityWithPut();
```

</TabItem>
</Tabs>

This will send a `PUT` request and include all existing properties on your entity, whether they were changed or not.
