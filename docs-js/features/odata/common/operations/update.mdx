The Update request builder allows you to send `PUT` or `PATCH` requests.
By default, `PATCH` is used to only update the changed fields:

<Tabs groupId="version" defaultValue="major" values={[ { label: 'SDK 2.x', value: 'major' }, { label: 'SDK 1.x', value: 'deprecated' } ]}>
<TabItem value="major">

```ts
const { businessPartnerApi } = businessPartnerService();
businessPartnerApi.requestBuilder().update(businessPartner);
```

</TabItem>
<TabItem value="deprecated">

```ts
BusinessPartner.requestBuilder().update(businessPartner);
```

</TabItem>
</Tabs>

In the example above only the changed fields of the given `businessPartner` instance are sent to the `BusinessPartner` service using `PATCH`.
