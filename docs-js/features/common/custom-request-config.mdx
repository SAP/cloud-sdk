By default, the SAP Cloud SDK uses [axios](https://github.com/axios/axios) as an HTTP client for executing requests.
The SAP Cloud SDK derives and configures most request options like `url`, `headers`, and others for you.
In certain cases, passing additional options to axios might be required.
To achieve this, you can provide a custom request configuration that will be passed down to axios.
